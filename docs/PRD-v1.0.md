## 一、产品概述

### 1.1 产品定位

随手记是一款轻量级的个人记账小程序，操作简单、无广告、注重隐私，让用户随时随地记录收支，培养良好的理财习惯。

**核心差异**：极简操作、3秒记账、无广告、隐私优先

### 1.2 核心需求回答

| 问题 | 答案 |
|------|------|
| 数据存储 | 本地优先 + 云端同步（多设备支持） |
| 多设备同步 | 需要，通过微信账号实现 |
| MVP功能 | 全部核心功能：快速记账、账单列表、月度统计、分类统计、预算管理 |
| 数据导出 | 需要，支持 Excel/CSV 格式 |

### 1.3 目标用户

| 用户群 | 占比 | 核心需求 |
|--------|------|---------|
| 月光族 | 30% | 控制支出、追踪消费 |
| 上班族 | 25% | 日常记账、月底复盘 |
| 学生党 | 20% | 生活费管理、培养习惯 |
| 家庭主妇 | 15% | 家庭收支记录 |
| 理财爱好者 | 10% | 精细化记账、数据分析 |

### 1.4 竞品分析

| 竞品 | 优点 | 不足 |
|------|------|------|
| 蚂蚁财富 | 功能全面、数据分析强 | 操作复杂、广告多 |
| 挖财 | 社区活跃 | 界面臃肿 |
| 随手记 | 简单易用 | 无微信小程序版 |

**我们的差异化**：微信原生体验、操作极简、无广告打扰、数据本地存储 + 云端同步

## 二、技术架构

### 2.1 技术栈选型

| 层级 | 技术 | 说明 |
|------|------|-------|
| 框架 | uni-app + Vue3 | 跨端开发框架 |
| UI组件库 | uView UI | Vue3组件库 |
| 状态管理 | Pinia | Vue3状态管理 |
| 本地数据库 | SQLite（uni-cloud-sqlite） | 本地数据持久化 |
| 云端同步 | 微信云开发 / 云数据库 | 多设备同步 |
| 图表库 | uni-echarts / uCharts | 统计图表 |
| 数据导出 | xlsx（前端Excel生成） | 导出功能 |

### 2.2 数据架构设计

**混合存储策略**：
- **主存储**：SQLite 本地数据库（离线可用、快速响应）
- **同步存储**：微信云数据库（多设备同步、备份）
- **同步机制**：本地优先，云端增量同步

```
┌─────────────────────────────────────────────┐
│                  用户设备                    │
│  ┌─────────┐    ┌─────────┐                │
│  │ 微信小程序 │◄──►│ SQLite  │                │
│  └─────────┘    └─────────┘                │
│        │                                    │
│        │ 增量同步                           │
│        ▼                                    │
│  ┌─────────────┐                           │
│  │ 微信云数据库 │  (多设备共享)               │
│  └─────────────┘                           │
└─────────────────────────────────────────────┘
```
